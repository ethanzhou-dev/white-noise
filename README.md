# WhiteNoise (白噪音)

用于播放背景白噪音的 Android 应用程序。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](./LICENSE)
![Android SDK](https://img.shields.io/badge/Android%20SDK-34+-3DDC84?logo=android&logoColor=white)
![Jetpack Compose](https://img.shields.io/badge/Jetpack_Compose-Material_3-4285F4?logo=android&logoColor=white)

## 功能

- **后台播放**: 支持应用在后台或设备熄屏状态下持续播放。
- **系统控制**: 可将其添加到控制中心的磁贴中进行快捷的播放/暂停操作。
- **定时关闭**: 提供倒计时功能，计时结束后自动停止音频。
- **状态记忆**: 本地记录并自动恢复上次使用的音效与倒计时设置。

## 安装

访问 [Releases](#) 页面下载最新版本的 `.apk` 文件，直接在 Android 设备上安装。

## 使用

- **播放音效**: 在首页列表点击对应的白噪音项目即可播放。
- **快捷控制**: 播放期间，添加磁贴后可通过控制中心的磁贴中进行播放/暂停。
- **设定定时**: 点击定时器图标输入时长，倒计时归零后自动停止播放。
